# Jefferson County, CO — Wildfire Ignition-Risk Viewer

Interactive map from the **ACE-FIRES** project (University of Washington) — a
data-driven *ignition-risk prior*: where wildfires are likely to **start**.

**[▶ Open the viewer](https://gilliamwong.github.io/jeffco-fire-viewer/)**

Toggle: ignition risk (annual + spring/summer/autumn), burnable fuel, recorded
fires (hover for cause), lightning strike density, trails & access points, and
comparison layers (a backtest map and USFS FSim burn probability).

Built from public data (LANDFIRE fuels, USGS 3DEP terrain, gridMET fire weather,
NLDN lightning, FPA-FOD/WFIGS fire records, USFS FSim, OpenStreetMap, SILVIS WUI,
Census/HIFLD). Held-out validation: human-ignition AUC ~0.79. **Research
prototype — not operational fire guidance.**
